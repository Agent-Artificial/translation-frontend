How to Use the Translation App

  1. Select your input mode (text or audio) from the dropdown menu.
  2. Choose your source language from the "From" dropdown.
  3. Choose your target language from the "To" dropdown.
  4. If using text input:
    • Type or paste your text into the input box.
  5. If using audio input:
    • Click the "Record" button to start recording.
    • Click "Stop" when you're done speaking.
    • Alternatively, upload an audio file using the "Upload" button.
  6. Click the "Translate" button to process your input.
  7. View the translated text or listen to the translated audio in the output section.
  8. Known issues:
    • Cannot reload the audio recorder after switching from audio to text and back.
    • File uploading is not fully implemented

For more detailed information, please refer to the README file in the project repository.
